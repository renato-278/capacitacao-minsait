Capacitação Minsait - API Spring Boot

Sobre o projeto

Este projeto é uma API REST desenvolvida em Java utilizando Spring Boot
como parte de um processo de capacitação técnica.

A aplicação realiza o gerenciamento de usuários e histórico de
transações, permitindo criar registros e consultar informações
armazenadas em banco de dados.

O objetivo do projeto é demonstrar a implementação de uma arquitetura
backend baseada em boas práticas, separando responsabilidades entre
controllers, services e repositories.

Tecnologias utilizadas

-   Java
-   Spring Boot
-   Maven
-   Banco de dados relacional
-   Spring Data JPA
-   API REST
-   Postman para testes


Regras de negócio implementadas
- Produtos 
- Carrinhos
- Usuários
- Historico de Transações
